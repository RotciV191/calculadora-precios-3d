# Calculadora de Precios 3D v7

App web para cotizar piezas de impresión 3D, guardar pedidos, clientes y ver dashboard de negocio.

## Nuevas funciones v7

- Nueva pestaña: Dashboard
- Ventas totales
- Ganancia estimada
- Saldo pendiente por cobrar
- Total de pedidos
- Total de clientes
- Anticipos recibidos
- Pedidos activos
- Próximas entregas
- Estado de pedidos
- Productos con más venta
- Alertas rápidas
