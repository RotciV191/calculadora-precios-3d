# Calculadora de Precios 3D

App web sencilla para cotizar piezas de impresión 3D.

## Cómo correr localmente

1. Instala Node.js.
2. Abre esta carpeta en VS Code o terminal.
3. Ejecuta:

```bash
npm install
npm run dev
```

## Cómo publicar en Vercel

1. Sube esta carpeta a GitHub.
2. Entra a Vercel.
3. Add New Project.
4. Importa el repositorio.
5. Framework: Vite.
6. Build Command: npm run build.
7. Output Directory: dist.
8. Deploy.

## Qué incluye

- Calculadora de costos
- Materiales base editables
- Materiales extra
- AMS/multicolor
- Postproceso
- Fallos
- Mayoreo
- Historial local en el navegador
